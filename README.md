# HDRV Cinematic Demo

**Can you believe Claude built this?**

Scroll-driven cinematic landing page for HDRV Studio. Built in one session with Claude.

## Features
- ✨ Animated canvas starfield
- 🎥 Film grain + vignette cinematic effects
- 🔮 3D rotating tunnel with particle dots
- 🧊 Glass morphism cards
- 📊 Scroll-triggered counter animations
- 🌊 Floating particle system
- 📱 Fully responsive
- ♿ prefers-reduced-motion support

## Deploy

```bash
vercel
```

Or serve locally:

```bash
python3 -m http.server 8080
```

## Built With
- Vanilla HTML/CSS/JS
- No frameworks, no dependencies
- Generated in one Claude session
